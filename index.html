About This Project

What is Big Pickle?

Big Pickle is a secure, zero-dependency project showcase. It lets you display open-source projects in a clean portfolio, with admin tools to manage them — all client-side, powered by GitHub.

Built With

  • Vanilla JavaScript (ES5 compatible)
  • Semantic HTML5
  • CSS Custom Properties (3 themes)
  • SHA-256 via Web Crypto API
  • GitHub REST API (Contents + Releases)
  • Service Worker (offline cache)

Author

serbodawg — open-source enthusiast, Linux tool builder.

github.com/serbodawg

License

MIT — free to use, modify, and distribute. No warranty.

Why This Exists

The Problem

Most project showcases rely on heavy frameworks, third-party backends, or SaaS platforms that track users, require accounts, and break on slow connections.

The Solution

A single HTML file that works on anything with a browser. No build step. No dependencies. No JavaScript framework. No tracking. Just HTML, CSS, and vanilla JS that runs on a potato.

Philosophy

  • Zero dependencies — every line is intentional
  • Security-first — protection is not an afterthought
  • Offline-capable — works without internet (once loaded)
  • Privacy-respecting — no cookies, no trackers, no analytics
  • Old-hardware friendly — runs on anything from 2010
  • GitHub-native — publish and sync via GitHub API

"The best tool is the one that stays out of your way and doesn't break when the internet goes down."
