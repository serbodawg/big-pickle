About Big Pickle

Our Mission

Big Pickle creates software that respects your freedom and your hardware. Every tool is designed to run fast on old machines, consume minimal RAM, and never phone home.

We believe good software should be boring — it should just work, stay out of your way, and not demand constant updates that break your workflow.

Open-Source Philosophy

All Big Pickle projects are released under permissive open-source licenses. You are free to inspect, modify, and distribute the source code.

No telemetry. No analytics. No cryptocurrency miners. No hidden agendas. The code is the documentation.

Technical Values

  • C / C99 — Systems-level languages for maximum performance.
  • Minimal dependencies — Every dependency is justified. Zero bloat.
  • X11 first — Born on X11, optimized for tiling window managers.
  • No JavaScript runtime — Tools are native executables, not Electron apps.
